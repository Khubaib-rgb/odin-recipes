# Odin Recipes

## Description
This is my first basic website project from The Odin Project foundation curriculum. The goal of this project is to build a simple recipes website using only foundational HTML. 

## Skills Demonstrated
By completing this project, I will have successfully demonstrated my understanding of:
* **Git & GitHub workflow:** Initializing, cloning, staging, committing, and pushing changes via the command line using SSH keys.
* **Core HTML syntax:** Structuring boilerplate text using proper tags.
* **Text formatting:** Utilizing headings (`<h1>` through `<h6>`), paragraphs, strong text, and emphasis.
* **Lists:** Building both ordered (`<ol>`) and unordered (`<ul>`) lists for ingredients and instructions.
* **Anchors & Links:** Linking internal pages together and linking out to external recipe sources.
* **File paths:** Properly organizing files and referencing relative paths within a project directory.
